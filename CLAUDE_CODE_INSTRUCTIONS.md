# 📋 Инструкция: Как залить проект на GitHub через Claude Code

## Что нужно перед началом

1. ✅ VS Code установлен
2. ✅ Claude Code установлен (расширение для VS Code)
3. ✅ Git установлен (скачать с https://git-scm.com если нет)
4. ✅ Аккаунт на GitHub (https://github.com — бесплатно)

## Шаг 1: Распакуй архив

Распакуй скачанный ZIP-архив в папку:
```
E:\Cars_Website\
```

Должна получиться структура:
```
E:\Cars_Website\
├── index.html
├── css/
├── js/
├── data/
├── README.md
└── ... остальные файлы
```

## Шаг 2: Открой в VS Code

1. Открой VS Code
2. File → Open Folder → выбери `E:\Cars_Website`

## Шаг 3: Запусти Claude Code

Открой терминал (Ctrl+`) и напиши:

```
claude
```

## Шаг 4: Дай команду Claude Code

Скопируй и вставь эту команду:

```
Инициализируй git-репозиторий в текущей папке.
Создай новый публичный репозиторий "car-deals" на GitHub.
Сделай первый коммит со всеми файлами и запушь на GitHub.
Затем включи GitHub Pages для ветки main (корень /).
Покажи мне финальную ссылку на сайт.
```

Claude Code выполнит всё автоматически. Если попросит авторизацию GitHub — следуй инструкциям в терминале.

## Шаг 5: Проверь сайт

После выполнения Claude Code даст тебе ссылку вида:
```
https://твой-логин.github.io/car-deals/
```

Открой её в браузере — сайт должен работать!

⚠️ GitHub Pages может включиться не сразу — подожди 1-2 минуты.

---

## Как потом вносить изменения

Когда хочешь что-то изменить:

1. Открой папку в VS Code
2. Запусти Claude Code (`claude` в терминале)
3. Опиши что нужно, например:
   - "Добавь новую машину в data/cars.json: Toyota Land Cruiser 2021, 4.5 млн, 30000 км"
   - "Измени цвет акцента на синий"
   - "Добавь фильтр по городу"
4. Claude Code внесёт изменения и запушит на GitHub
5. Через 1-2 минуты изменения появятся на сайте

## Частые проблемы

**"git не найден"**
→ Установи Git: https://git-scm.com/download/win

**"Нет доступа к GitHub"**
→ Claude Code попросит войти в GitHub — следуй подсказкам

**"Сайт не открывается"**
→ Зайди на GitHub → твой репозиторий → Settings → Pages → убедись что Source = "Deploy from branch", Branch = "main", Folder = "/ (root)"

**"Данные не загружаются"**
→ Проверь что файл data/cars.json существует и содержит валидный JSON
