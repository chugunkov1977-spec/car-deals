## 🎨 Дизайн (v2+)
- **Тема**: тёмная, премиальная
- **Акцент**: золотой (#c3a56e)
- **Шрифты**: Playfair Display (заголовки) + Onest (тело)
- **Карточки**: градиентный фон (уникальный для марки) + фото машины поверх + drop-shadow
- **Эффекты**: backdrop-filter blur, stagger-анимации, градиенты

## 📈 Алгоритм выгодности
Deal score 0-100:
1. Цена vs средняя для марки/модели (до +30)
2. Цена за 1000 км vs средняя в группе (до +15)
3. Абсолютная цена за 1000 км (до +10)
4. Свежесть — новые с малым пробегом (до +5)

Уровни: 80+ hot, 60+ good, 55+ isDeal, 40+ ok, <40 normal

## 🖼 Система фото (v5)
Порядок загрузки изображения:
1. Локальный файл images/cars/{Brand}_{Model}.png (35 из 40 моделей)
2. Если нет → car.image из JSON (внешняя ссылка)
3. Если и она не грузится → SVG-силуэт с названием марки и типом кузова
