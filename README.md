# 🚗 AutoDeal — Выгодные авто России

Сайт для анализа и поиска выгодных объявлений о продаже автомобилей на российском рынке.

## Что умеет сайт

- **Анализ выгодности** — каждое объявление получает индекс от 0 до 100
- **Два критерия**: цена ниже рыночной + лучшее соотношение цена/пробег
- **Фильтры** — по марке, году, цене, пробегу, кузову, приводу
- **Сортировка** — по выгодности, цене, пробегу, году
- **Мобильная версия** — работает на телефоне
- **Тёмная тема** — современный дизайн

## Как запустить локально

1. Откройте папку проекта в VS Code
2. Установите расширение **Live Server** (если нет)
3. Нажмите правой кнопкой на `index.html` → **Open with Live Server**
4. Сайт откроется в браузере

## Как залить на GitHub (через Claude Code)

Откройте терминал в VS Code (`` Ctrl+` ``) и запустите Claude Code. Скажите ему:

## Как обновлять данные

Все объявления хранятся в файле `data/cars.json`. 

Каждое объявление — это объект с полями:
- `brand` — марка (Toyota, Kia и т.д.)
- `model` — модель (Camry, Rio и т.д.)
- `year` — год выпуска
- `price` — цена в рублях
- `mileage` — пробег в км
- `engine` — двигатель
- `power` — мощность (л.с.)
- `transmission` — коробка (AT/MT/CVT/DSG/DCT)
- `drive` — привод (FWD/RWD/AWD)
- `body` — кузов (Седан/Кроссовер/Лифтбек)
- `city` — город
- `source` — источник (avito/auto.ru)
- `source_url` — ссылка на оригинал
- `image` — ссылка на фото
- `description` — описание
