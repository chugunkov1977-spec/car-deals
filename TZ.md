# Техническое задание: AutoDeal — Поиск выгодных автомобилей

## 1. Цель проекта
Создать веб-сайт, который показывает самые выгодные объявления о продаже машин на российском рынке (Avito, Auto.ru). Сайт анализирует объявления и выделяет те, где цена ниже рыночной или соотношение цена/пробег лучше среднего.

## 2. Критерии «выгодности»
- **Цена ниже рыночной**: сравниваем цену объявления со средней ценой для той же марки/модели/года. Если цена на 15%+ ниже средней — помечаем как выгодное.
- **Лучшее соотношение цена/пробег**: считаем «стоимость за 1000 км» = цена / (пробег / 1000). Чем меньше — тем выгоднее. Сравниваем с аналогами.

## 4. Функционал сайта

### 4.1 Главная страница
- Шапка с логотипом и названием «AutoDeal»
- Блок статистики: сколько объявлений, средняя цена, лучшая сделка дня
- Карточки объявлений с фото, ценой, пробегом, маркой, годом
- Бейдж «ВЫГОДНО» на карточках, прошедших анализ

### 4.2 Фильтры
- По марке (выпадающий список)
- По году выпуска (от — до)
- По цене (от — до)
- По пробегу (от — до)
- Только выгодные (чекбокс)
- Сортировка: по выгодности, по цене, по пробегу, по дате

### 4.3 Карточка объявления (при клике)
- Полная информация о машине
- График: сравнение цены с рыночной средней
- Оценка выгодности (индикатор)
- Ссылка на оригинал объявления (Avito/Auto.ru)

### 4.4 Аналитика
- Средняя цена по маркам/моделям (автоматический расчёт из данных)
- Индекс выгодности для каждого объявления
- Топ-10 самых выгодных сделок

## 5. Дизайн
- Тёмная тема (автомобильная тематика)
- Акцентный цвет: яркий зелёный (#00ff88) — «выгода», «деньги»
- Стиль: современный, чистый, информативный
- Мобильная версия обязательна (responsive)
